## Requirements

`ionic/cli`

`npm or yarn`

`visual studio for build`


## Installation

Clone this repository : 

`https://github.com/julkwel/signaleo.git`

Go inside

`cd signaleo`

Install dependencies

`npm install`

## Running

Run server

`ionic serve`

For device 

`ionic capacitor run android`

## Build

For build run :

`ionic build`
